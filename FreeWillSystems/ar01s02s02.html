2.2. Change

Consider the following:

  • Several readings of atmospheric pressure at different altitudes.

  • Several moves of the white pieces of a chess play.

  • Different colors preffered by individual persons in a group.

  • Different positions of an object that is moving around.

Each of the items in that list is a form of Sampling: that is, a collection of values each associated with a marker, where each (value, marker) pair is a sample.[1] The above list can be presented as the following collections of example samples:

  • {(101.325kPa, 0m),(80kPa, 2000m),(60kPa, 4000m)}

  • {(Be5, 1),(Nf3, 2)}[2][3]

  • {(red, Bob),(blue, Alice)}

  • {(point0, instant0),(point1, instant1)}


[1] The concepts of Sampling are presented here in a way that fits the purpose of this work, but the reader should know that if differs somehow from the formal Sampling Theory of Statistics

[2] The value of the samples in these example pairs are chess moves using the so called Standard Algebraic Notation for chess

[3] In this case, the marker is an index to the step along the play
